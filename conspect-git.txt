Задать имя пользолвателя: git config --global user.name "Kaiyr Nurasyl"
Задать почту пользолвателя: git config --global user.email "kair.nurasyl@gmail.com"
Инициялизация пустую локальную репозиторию: git init
Соединение локальную репозиторию с удаленный реп.: git remote add origin git@github.com:nurasylkaiyr/test.git
Статус: git status
Добавить определенные файлы для коммита: git add .readme.txt
Коммит: git commit -m "message"
Информация об коммитах: git log
Вернуться к определенному коммиту: git checkout 0a1fa6878e481e83b91e48cdf2c4f2b3b969da20
Опубликовать коммиты на удаленную репозиторию: git push origin2{имя репозитория} master{ветка}
Скачивает все изменений из удаленную на локальную репозиторию: git pull origin2{имя репозитория} master{ветка}
Информация о ветках: git branch
Создание новой ветки: git branch feature{имя новой ветки}
Переход между ветками: git checkout master{имя ветки}
Производит слияние данной ветки в ветку, на которой прямо сейчас находится пользователь в виде одного нового коммита в текущую ветку: git merge feature{имя ветки}
